## Технологии

- Python 3.11 для основной логики бота.
- uv для управления зависимостями и изоляции окружения (`uv init`, `uv run`, `uv pip`).
- make как единая точка входа в команды (`make install`, `make run`, `make lint`, `make test`).
- aiogram 3 с polling для работы с Telegram Bot API.
- openai python client для доступа к LLM через OpenRouter.
